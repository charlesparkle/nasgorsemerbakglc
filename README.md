# 🍛 Nasgor Semerbak GLC - Premium Landing Page

Landing page modern, premium, dan SEO-optimized yang dirancang khusus untuk brand **Nasgor Semerbak** di Green Lake City (GLC). Proyek ini difokuskan pada peningkatan kredibilitas digital, konversi pesanan via platform *online* (GoFood/GrabFood), dan otomatisasi pesanan *Bulk Order* melalui WhatsApp.

## ✨ Fitur Utama (Key Features)

- **Premium UI/UX:** Desain *mobile-first* dengan palet warna hangat (Merah Rempah, Cokelat Kayu, aksen Gold), efek *glassmorphism* pada navigasi, dan *subtle hover animations*.
- **SEO & Local Search Optimized:** Terintegrasi dengan Meta Tags lengkap, Open Graph, dan **Schema Markup (JSON-LD)** bertipe `Restaurant` dan `WebSite` untuk mendominasi pencarian "Nasgor Green Lake City" dan memastikan nama *brand* terbaca benar oleh Google.
- **Dynamic WhatsApp Ordering:** Fitur "Racik Seleramu" yang interaktif, memungkinkan pelanggan memilih *topping*, level pedas, dan opsi sayuran, yang secara otomatis di-*generate* menjadi pesan WhatsApp siap kirim untuk kebutuhan *Bulk Order Bukber*.
- **Performance Focused:** Dibangun murni dengan HTML5, CSS3, dan Vanilla JavaScript tanpa *framework* berat untuk memastikan waktu *loading* super cepat (Crucial for Local SEO).
- **Interactive Menu:** Tampilan menu yang elegan, dibagi menjadi *grid clean* untuk menu reguler dan *visual cards* eksklusif dengan efek *zoom hover* untuk menu premium (Sapi Asap, Ayam Asap, dll).

## 🛠️ Teknologi yang Digunakan (Tech Stack)

- **HTML5:** Struktur semantik, aksesibilitas, dan injeksi *metadata/schema*.
- **CSS3:** Custom properties (variabel global), Flexbox, CSS Grid, animasi kurva *bezier* kelas atas, dan *media queries* untuk responsivitas layar penuh.
- **Vanilla JavaScript (ES6+):** Logika *hamburger menu mobile*, Intersection Observer untuk animasi *scroll* (fade-in), dan manipulasi DOM untuk *dynamic form* ke WhatsApp.

## 🚀 Cara Menjalankan (How to Run Locally)

Karena proyek ini menggunakan teknologi web murni tanpa proses *build/compile*, Anda bisa langsung menjalankannya dengan mudah:

1. *Clone repository* ini ke komputer lokal Anda.
2. Buka folder proyek di Code Editor Anda (disarankan VS Code/Cursor).
3. Buka file `index.html` langsung di *browser*, **ATAU** gunakan ekstensi **Live Server** untuk pengalaman *development* dengan *hot-reload*.

## 🌐 Deployment

Proyek ini telah dikonfigurasi dan siap untuk di-*deploy* langsung ke platform *hosting* modern. Saat ini, versi *live* dijalankan menggunakan integrasi otomatis dengan **Vercel** (`nasgorsemerbak.vercel.app`).
